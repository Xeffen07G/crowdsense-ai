AGENT ROSTER:
  1. Planner Agent    вҶ’ Intent parsing, coordination, response assembly
  2. Navigation Agent вҶ’ Crowd-aware pathfinding and route optimization
  3. Queue Agent      вҶ’ Wait time prediction and service point ranking
  4. Crowd Agent      вҶ’ Real-time crowd density simulation and analysis
  5. Experience Agent вҶ’ Smart recommendations and experience enhancement

WORKFLOW:
  1. User submits a query (e.g., "Find me food with least wait time")
  2. Planner Agent classifies intent and activates relevant agents
  3. Crowd Agent provides current density data to other agents
  4. Navigation Agent computes optimal route avoiding congestion
  5. Queue Agent estimates wait time based on crowd level
  6. Experience Agent generates tailored recommendations
  7. Planner Agent aggregates all outputs into structured response
  8. UI renders the response in premium card format

DESIGN PRINCIPLES:
  - Privacy First: No personal data is stored or transmitted.
  - Real-time: All outputs reflect current simulated conditions.
  - Actionable: Every response includes a clear next step.
  - Accessible: UI is responsive and works on all devices.
  - Delightful: Premium design with smooth animations and rich visuals.

TECHNOLOGY:
  - Frontend: Vanilla HTML5, CSS3, JavaScript (ES6+)
  - Architecture: Client-side multi-agent simulation
  - Deployment: Static hosting (GitHub Pages compatible)
  - Dependencies: None (zero external libraries)
